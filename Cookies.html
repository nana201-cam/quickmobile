Gestion des cookies – MYFIX QUICK MOBILE FIX

Lors de votre navigation sur notre site https://myfix.fr, des cookies peuvent être déposés sur votre navigateur, uniquement pour garantir le bon fonctionnement du site et mesurer de manière anonyme l’audience. Chaque navigateur web propose des réglages spécifiques pour gérer les cookies. La configuration de ces paramètres dépend du logiciel que vous utilisez pour naviguer sur Internet. Nous vous invitons à consulter le menu d’aide de votre navigateur pour savoir comment modifier vos préférences en matière de cookies. Voici quelques liens directs vers les instructions de gestion des cookies pour les navigateurs les plus courants : Pour Firefox Pour Google Chrome Pour Internet Explorer Pour Safari Pour Opera

Pour plus d’informations sur les cookies, nous vous invitons à consulter le site de la CNIL : https://www.cnil.fr/fr/cookies-les-outils-pour-les-maitriser
